# sequelize-poc

Start the databases.

```bash
npm run database:up
```

Stop the databases.

```bash
npm run database:down
```

Run all tests, coverage, and lint.

```bash
npm test
```

Run Mocha without coverage.

```bash
npm run test:dev

```

Run Mocha with watch during development.

```bash
npm run test:dev:watch
```

Run EsLint.

```bash
npm run lint
```

Run test coverage.

```bash
npm run test:coverage
```
